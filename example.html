Drift Example - Overlapping
This example immediately modifies / hides the drift menu on activation / deactivation of anchors. Because of this, it requires the drift element to be positioned overlapping the anchor to be usable. Otherwise the drift element will disappear / its contents will change when anchors deactivate as you try to mouse over to it. For an example of a setup that does not require this overlapping, see "Drift Example - Free Floating". Press "x" to temporarily hide / unhide the drift element. Press "x+z" to permanently hide / unhide it.
