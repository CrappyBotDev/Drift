Drift Example - Free Floating
This example is set up to separate the drift element updating and the anchors deactivating so that the user has the opportunity to mouse over to the drift element before it updates / disappears. There's a 3 second delay to the drift element updating that gets paused / reset when interacting with the drift element. This is just one way this could be done, see "example_free_float.js" for implementation details.
